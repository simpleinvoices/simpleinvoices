# Simple Invoices

Welcome to the Simple Invoices documentation. Simple Invoices is a free, open-source web application for creating and managing invoices, clients, and payments.

**Version:** 2013.1.beta.8

## Need Help?

Browse the sidebar for detailed guides, or use the **search** bar above to find specific topics. Context-sensitive help is available throughout the application — look for the <kbd>?</kbd> icon next to fields and settings.

## Key Features

- **Invoice Management** — Create itemized or total-style invoices
- **Multi-Currency** — Support for 40+ currencies with automatic tax calculation
- **Payment Tracking** — Record payments, process online payments via gateways
- **Customer & Biller Management** — Maintain detailed client and vendor records
- **Product Inventory** — Track products, attributes, and stock levels
- **Reports** — Generate sales, debtor, and profit reports with charts
- **Multi-Language** — Available in 41+ languages
- **PDF Export** — Generate professional PDF invoices and statements
- **Email Integration** — Send invoices directly via SMTP
- **Custom Fields** — Extend invoices and customers with your own fields
