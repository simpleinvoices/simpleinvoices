# Web-Based & Mobile Friendly

Simple Invoices is a **web application**: access it from any device with a browser. No app installation required.

## Web-Based Architecture

- Runs entirely in your browser (no desktop app needed)
- Zero client-side installation
- Works on any operating system: Windows, macOS, Linux
- Accessible from anywhere on your network

## Mobile Experience

The interface is built with **Tabler CSS**, a responsive Bootstrap-based framework:

- Adaptive layout that works on phones, tablets, and desktops
- Touch-friendly controls and navigation
- Readable invoice views on small screens
- Full functionality on mobile (create, edit, send invoices)

## What You Can Do on Mobile

| Task | Mobile Support |
|------|---------------|
| Create invoices | Yes |
| View invoice list | Yes |
| Edit invoices | Yes |
| Email invoices | Yes |
| Record payments | Yes |
| View reports | Yes |
| PDF download | Yes |
| Customer management | Yes |

## Responsive Design

The interface adapts automatically:

- **Desktop**: Full sidebar navigation, multi-column layouts
- **Tablet**: Collapsed sidebar, optimised forms
- **Phone**: Stacked layouts, simplified navigation

## Technology

| Component | Technology |
|-----------|-----------|
| **CSS Framework** | Tabler (Bootstrap 5 based) |
| **Icons** | Tabler Icons (3,000+ SVG icons) |
| **Interactivity** | jQuery + vanilla JavaScript |
| **Charts** | ApexCharts (responsive SVG charts) |
| **Date Pickers** | Litepicker (touch-friendly) |
| **Select Inputs** | Tom Select (enhanced dropdowns) |
