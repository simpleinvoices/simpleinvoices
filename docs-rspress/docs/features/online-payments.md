# Online Payments

Accept payments directly through your invoices with integrated support for **9 payment gateways**.

## Supported Gateways

| Gateway | Type | Region |
|---------|------|--------|
| **Stripe** | Credit/debit cards, wallets | Global |
| **PayPal Commerce** | PayPal, cards, local methods | Global |
| **Mollie** | iDEAL, Bancontact, SEPA, cards | Europe |
| **Authorize.net** | Credit/debit cards, eCheck | US/Canada |
| **eWay Rapid** | Credit/debit cards | Australia/NZ |
| **Payments Gateway** | Multi-processor gateway | Global |
| **Ko-fi** | Donations, tips, one-off payments | Global |
| **Coinbase Commerce** | Bitcoin, Ethereum, USDC, crypto | Global |
| **Adyen** | Cards, local methods, wallets | Global |

## How It Works

1. Enable gateways under **Settings → Invoice Preferences → Payment tab**
2. Configure each gateway's API credentials under **Settings → Payment Gateways**
3. Select which gateways to offer per invoice preference
4. Customers see pay buttons/links directly on their invoices

## Payment Flow

1. Customer receives invoice via email or portal
2. Clicks the payment link or button for their preferred gateway
3. Completes payment on the gateway's secure page
4. Simple Invoices records the payment and marks the invoice as paid

## Multi-Gateway Per Invoice

You can enable multiple gateways on a single invoice preference. Customers choose how to pay.

## Cryptocurrency Support

Coinbase Commerce integration enables crypto payments with:

- Bitcoin (BTC)
- Ethereum (ETH)
- USDC (stablecoin)
- Litecoin (LTC), Dogecoin (DOGE), and more

Crypto payments are automatically confirmed on-chain.
