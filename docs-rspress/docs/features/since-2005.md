# Since 2005: 20+ Years of Invoicing

Simple Invoices has been helping businesses get paid since **2005**. That's over two decades of continuous development, community feedback, and real-world invoicing.

## Timeline

| Year | Milestone |
|------|-----------|
| **2005** | First release. Built as a simple PHP invoicing tool |
| **2007** | Multi-language support, PDF generation |
| **2009** | Multi-currency, custom fields, payment tracking |
| **2011** | Version 2011.1: stable release, wide adoption |
| **2013** | Version 2013.1 beta: modernized codebase |
| **2020** | Zend Framework 1 Future fork for PHP 7+ compatibility |
| **2023** | Tabler UI rewrite, modern responsive design |
| **2024** | PHP 8.1+ full compatibility, PHPMailer 6 upgrade |
| **2025** | Laravel Blade templates, multi-domain SaaS architecture |
| **2026** | Docker support, Rspress documentation, 41+ languages |

## Stability

- Battle-tested by thousands of users over 20 years
- Proven data integrity: no corruptions, no lost invoices
- Backward compatibility: archives from 2005 still readable
- Active maintenance with regular updates

## Why Longevity Matters

Choosing invoicing software is a long-term decision:

- **Data ownership**: Your 2005 invoices are still in your database today
- **No rug pulls**: Open-source means the project can't disappear overnight
- **Institutional knowledge**: The codebase has been refined through real-world feedback
- **Community trust**: Thousands of users depend on it daily

Simple Invoices is one of the longest-running open-source invoicing projects on the web.
