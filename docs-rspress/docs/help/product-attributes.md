# Product Attributes

Product attributes allow you to define variations of your products (e.g., sizes, colors, formats).

## Overview

Attributes are useful for:

- Products that come in multiple variants (S/M/L)
- Services with different tiers (Basic/Premium)
- Items with configurable options (Color, Material)

## Setting Up Attributes

### Step 1: Define Attribute Types

1. Go to **Products → Attributes**
2. Click **New Attribute**
3. Enter the attribute name (e.g., "Size", "Color", "Format")
4. Save

### Step 2: Define Attribute Values

1. Go to **Products → Values**
2. Select the attribute type
3. Add values (e.g., "Small", "Medium", "Large")
4. Save

### Step 3: Assign to Products

When creating or editing a product, select the appropriate attribute values:

1. Open a product
2. In the Attributes section, select the attribute and its value
3. A product can have multiple attributes
4. Save the product

## Using Attributes on Invoices

When you add a product with attributes to an invoice:

- The attribute values are displayed alongside the product description
- Attributes help distinguish between product variants in reports
- Inventory tracking (if enabled) is managed per attribute combination

## Example Configuration

**T-Shirt Product:**
- Attribute: "Size" → Values: S, M, L, XL
- Attribute: "Color" → Values: Red, Blue, Black

**Software License:**
- Attribute: "Type" → Values: Basic, Professional, Enterprise

**Consulting Service:**
- Attribute: "Duration" → Values: 1 Hour, Half Day, Full Day
