# Welcome to the New Simple Invoices

We're excited to announce a major refresh of Simple Invoices: the open-source invoicing platform trusted by businesses worldwide since 2005.

## What's New

### Modern UI with Tabler

The entire interface has been rebuilt with **Tabler CSS**, a modern Bootstrap 5-based framework. The result: a cleaner, faster, and more responsive experience on desktop, tablet, and mobile.

### Laravel Blade Templates

Invoice templates now use **Laravel Blade**: the same templating engine used by millions of Laravel developers. This means easier customisation and better performance.

### PHP 8.1+ Compatibility

Full compatibility with modern PHP versions (8.1, 8.2, 8.3, 8.4). The Zend Framework 1 Future fork and PHPMailer 6 ensure your stack stays current.

### Docker Deployment

Get up and running in minutes with the new Docker image. Includes PHP-FPM, Nginx, and support for MySQL, MariaDB, PostgreSQL, and SQLite.

### Multi-Domain SaaS

Run multiple isolated invoice domains from a single installation. Perfect for SaaS platforms, multi-brand organisations, and regional offices.

### Rspress Documentation

The new documentation site (you're looking at it!) is built with Rspress: a modern static site generator with search, RSS feeds, and a responsive design.

## What's Next

- Expanded API for integrations
- Enhanced reporting and analytics
- More payment gateway integrations
- Improved multi-language coverage

Stay tuned for more updates. As always, Simple Invoices remains **free and open-source** under the GPLv3 license.
